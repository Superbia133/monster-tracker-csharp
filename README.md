# Monster Tracker

## Description

Monster Tracker is a C# console application designed for tabletop role-playing games. It allows users to manage monsters during encounters by tracking health, abilities, attacks, damage types, weaknesses, and conditions.

## Features

- Add monsters
- View monster summaries
- View detailed monster information
- Add abilities
- Add attacks
- Add damage types
- Add weaknesses
- Add conditions
- Damage monsters
- Weakness damage multiplier
- Heal monsters
- Remove monsters
- Save and load encounters using JSON

## Technologies Used

- C#
- .NET Console Application
- System.Text.Json

## Future Improvements

- Resistances and immunities
- Initiative tracking
- Dice rolling system
- Graphical user interface
- Encounter builder

## How to Run

1. Clone the repository.
2. Open the project in Visual Studio Code.
3. Open a terminal.
4. Run:

```bash
dotnet run
```
